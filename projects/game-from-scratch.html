A small browser adventure game built from first principles.

Cool Adventure Game

I set out to build a game from scratch, out of curiosity for the basics of game engines and how developers of early games would build an engine from scratch for the purposes of each game. I ended up with a fully functional game made only from JavaScript and the Canvas API. With this repo I paid special attention to the cleanliness and structure of the code to make future development easier, spending many hours on refactoring. There are still levels and graphics to add, not to mention sound, but it is a fun lesson in game development and fully playable for now. Until it is complete, it can be played by cloning the repo and running it as a python script as described in the readme.
